# Пакет для подачи в SMS.RU (согласование имени и шаблонов)

> Что подаём через личный кабинет SMS.RU (не операторам напрямую — SMS.RU
> сам передаёт имя и шаблоны на модерацию операторам МТС/Билайн/Мегафон/Теле2).
> Код всегда 4 цифры, срок действия — 10 минут. Всё сервисное, без рекламы.

## 1. Имя отправителя (Sender ID)
- Желаемое имя: `ZaVed`
- Можно на старте оставить ПУСТЫМ — тогда SMS уходят от общего имени без абонплаты.
- Имя согласуется отдельно у каждого оператора (у каждого своя абонплата).

## 2. Тексты SMS-шаблонов
Переменная часть (код) обозначается как `{{code}}` (реальный код подставляет система).

### Шаблон 1 — Вход в приложение
```
Kod dlya vhoda v ZaVeduyushchaya: {{code}}. Nikomu ne soobshchayte.
```
Кириллица (альтернатива):
```
Код для входа в ЗаВедующая: {{code}}. Никому не сообщайте.
```

### Шаблон 2 — Регистрация / первый вход с нового устройства
```
Код подтверждения регистрации в ЗаВедующая: {{code}}. Действует 10 минут.
```

### Шаблон 3 — Подписание документа по ПЭП (универсальный)
Покрывает: согласие на ПДн, договор, оферту, акт, товарную накладную.
```
Код подписания документа №{{doc_number}}: {{code}}. Ввод кода = подпись ПЭП.
```
Вариант с типом документа (если нужно показать клиенту, что он подписывает):
```
Код подписания: {{doc_title}} №{{doc_number}} — {{code}}. Ввод кода = подпись ПЭП.
```
- `{{doc_number}}` — сквозной номер документа, формат `2026-0001`.
- `{{doc_title}}` — «Акт выполненных работ» или «Товарная накладная» (из doc_type).
- Форму документа (simple/ТОРГ-12/УПД) в SMS НЕ указываем — она влияет только на PDF.

## 3. Реквизиты компании (если запросит SMS.RU при регистрации имени)
- ООО «ЗаВедующая» — ИНН и ОГРН юрлица.

## 4. Назначение рассылки (формулировка для модерации)
```
Сервисные (транзакционные) сообщения: коды авторизации при входе и регистрации,
а также коды подписания документов простой электронной подписью (ПЭП).
Реклама и массовые рассылки отсутствуют.
```

## Примечания
- Латиница дешевле и надёжнее доходит (140 символов против 70 кириллицей).
- Для входа рекомендуется латиница, для подписания — кириллица (юридически понятнее).
- Точный синтаксис переменных и перечень требуемых документов — уточнить
  в поддержке SMS.RU при регистрации имени отправителя.
